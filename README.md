
# 🌐 Deekshitha Puppala — Personal Portfolio (Frontend)

This repository contains the frontend for my personal portfolio website.  
The portfolio showcases my projects, skills, and learning journey while demonstrating modern frontend development practices.

The goal of this project was not just to create a portfolio, but to build a **real full-stack system** with a working backend, database, and admin dashboard.

# ✨ Features

• Modern responsive portfolio design  
• Smooth animations using Framer Motion  
• Scroll-based navigation  
• Resume preview and download  
• Contact form connected to backend API  
• Dark / Light theme support  
• Mobile-first responsive layout

Install dependencies

npm install

Start development server

npm run dev
